# ScrcpyGUI (v1.4) - ユーザーガイド

シンプルなUIで、scrcpyの全機能を最大限に活用できるGUIツールです。

## 🚀 最新アップデート (v1.4)
ユーザーの皆様からのフィードバックに基づき、大規模な改善を行いました。

### 1. 🔧 トラブルシューティング（ジェスチャー不具合対策）
接続終了後にスマホの「戻る」や「ホーム」が効かなくなる問題に対応しました。
- **自動修正**: 仮想ディスプレイ終了時に、自動的に設定を元に戻す処理が走ります。
- **完全リセット**: 「デバイス設定を強制リセット」ボタンで、ジェスチャー設定だけでなく、解像度や密度(`wm size/density`)も初期状態に戻します。

### 2. 📝 プロフェッショナルUI
アイコンや絵文字を完全に排除し、ビジネス用途でも使いやすい「文字ベース」のUIに統一しました。

### 3. ✅ scrcpy全機能の完全網羅
UI上で設定できる項目を大幅に増やしました。
- **ウィンドウ設定**: 起動位置 (X/Y)、サイズ、ボーダーレス（枠なし）起動に対応。
- **入力設定**: ショートカットキー設定、操作無効化、レガシーペーストモード。
- **その他**: 画面回転ロック、クリップボード同期設定など。

### 4. ⌨️ カスタム引数
UIにないマニアックなオプションも**「追加引数」**欄に直接入力して利用可能です。

### 5. 🔊 仮想ディスプレイの音声転送
仮想ディスプレイ使用時も、PCからスマホの音を出せるようになりました（Android 11+）。

## 📖 使い方

### 1. 基本操作 (ミラーリング)
1. **[更新]** ボタンでデバイスリストを読み込みます。
2. クリックしてデバイスを選択します。
3. **[ミラーリング開始]** をクリックします。

### 2. 仮想ディスプレイ (拡張画面)
PCのサブモニターとしてAndroidを使いたい場合に最適です。
1. **[仮想ディスプレイ]** タブを選択します。
2. **パフォーマンスモード**を選択します。
   - **最高画質**: 有線接続向け (2K/120fps/20Mbps)
   - **バランス**: 標準設定 (FHD/60fps/8Mbps)
   - **低遅延**: 無線向け (720p/30fps/4Mbps)
3. **詳細設定** (New!):
   - **最大FPS**: 30/60/90/120fps または制限なしを選択可能。
   - **ビットレート**: 4M～20Mまで調整可能。
   - **コーデック**: H.265 (推奨) / H.264 / AV1。
4. **[仮想ディスプレイを開始]** をクリックします。

### 3. 画面録画
1. **[録画]** タブで「画面録画を有効にする」にチェック。
2. 保存先を選んで、ミラーリングまたは仮想ディスプレイを開始すれば自動的に録画されます。

## 📥 配布と起動
- **最新版**: `dist_v1.3.0` フォルダ (※v1.4のコード変更を含みます)
- **起動コマンド**: `npm start`
